# CGAAT-Inspired Flask Website

A modern white-theme Flask website inspired by CGAAT with:
- same-style home page content
- register and login
- user dashboard
- admin panel
- 3 test packages with different questions
- blog with working pagination
- working footer navigation
- SQLite database

## Run locally

```bash
pip install -r requirements.txt
python app.py
```

Open: `http://127.0.0.1:5000`

## Features
- Home page inspired by CGAAT content
- 3 tests: Aptitude, Personality, Career Guidance
- Different package for each test
- User can register, login, activate package, take test, and view results
- Admin can view users, test attempts, and contact messages
