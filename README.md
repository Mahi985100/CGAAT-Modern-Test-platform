# CGAAT Modern Test Platform

A modern Flask-based assessment platform with:
- User & Admin Panel
- 3 Different Exam Types
- Pattern & IT-based Questions
- Result Generation
- Clean UI with Hover Effects

## Tech Stack
- Python (Flask)
- HTML, CSS, JS

## Features
- Login / Register
- Admin Dashboard
- User Dashboard
- Online Test System
- Result Calculation

## How to Run
1. Install dependencies
pip install -r requirements.txt

2. Run app
python app.py

3. Open browser
http://127.0.0.1:5000
